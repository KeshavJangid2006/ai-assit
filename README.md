# 💡 How It Works

1. Listens to the user's voice.
2. Converts speech into text.
3. Checks whether the command is predefined.
4. Executes built-in functions for known commands.
5. Unknown commands are sent to Google Gemini AI.
6. Gemini generates an intelligent response.
7. Jarvis speaks the response using text-to-speech.
